# Frontend Driven Development Marketplace

Claude Code plugin marketplace for [frontend-driven-development](https://github.com/Jordyrock15/frontend-driven-development).

## Installation

```bash
# Add the marketplace
/plugin marketplace add Jordyrock15/frontend-driven-development-marketplace

# Install the plugin
/plugin install frontend-driven-development
```

## What's Included

- **frontend-driven-development** — 10 composable skills that enforce disciplined frontend development: senior engineer standards, codebase scanning, architecture planning, visual verification, strict TypeScript, security, testing, framework patterns, task orchestration, and code review.
