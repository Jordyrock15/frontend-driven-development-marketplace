# Frontend Driven Development Marketplace

Claude Code plugin marketplace for [frontend-driven-development](https://github.com/Jordyrock15/frontend-driven-development).

## Installation

```bash
# Add the marketplace
/plugin marketplace add Jordyrock15/frontend-driven-development-marketplace

# Install the plugin
/plugin install frontend-driven-development
```

## What's Included

- **frontend-driven-development** — 12 composable skills that enforce disciplined frontend development: engineering standards, codebase scanning, architecture planning, task orchestration with sub-agent dispatch, workflow execution, strict TypeScript, framework patterns, testing, security, error recovery, bundle analysis, and git workflow.
